cf.IP 특징 : IP는 사실 32자리로 이루어진 2진수, IP는 네트워크 영역과 호스트 IP로 구성됨,   

​                    동일한 네트워크 내에서 호스트 IP는 각자 달라야 함.    

cf.서브넷마스크 특징 :하나의 IP에서 사용자가 필요한 IP만쓰게, 네트워크 영역과 호스트 영역을 나눠준다. 기본 서브넷 네트워크도 물론 네트워크 영역과 호스트 영역을 나눠주기 때문에 서브넷 마스크라고도 불림.
